# Project Log

## 2026-06-16

当前目标：设计一个平台无关的应届生简历教练 agent，用于指导应届大学毕业生制作求职简历。

关键决策：

- 第一版交付为平台无关的提示词/工作流方案，不先做网页或 API。
- Agent 同时覆盖从零生成简历、优化已有简历、针对岗位 JD 定制简历三类任务。
- 采用“主 agent + 模块化规则 + 可扩展知识库”方案。
- 学生分类采用“求职方向 + 学生背景”双维度。
- 输出采用“诊断报告 + 可投递简历 + 修改逻辑说明”。
- 指导风格采用“教练式挖掘 + HR 式打磨”。
- 真实性边界为“适度包装表达，但不编造事实”。
- 支持中文、英文和中英双语，默认中文。
- 知识库第一版先设计可扩展结构和少量样例。
