# 测试案例：考公国企方向

## 用户输入

我想投国企管培或事业单位综合岗。本科期间担任过班长，组织过班级团日活动和就业信息收集；写过几次活动总结和通知；做过 40 小时志愿服务。希望简历表达稳重一点。

## 期望 agent 行为

Agent 应识别为公务员、国企、事业单位方向，使用稳重、规范、克制的表达。Agent 应优先挖掘组织协调、材料写作、责任意识、志愿服务和服务对象，避免互联网营销话术。

## 必须出现

- 对目标方向的识别：国企、事业单位或公共部门方向。
- 关于班长职责、活动次数、参与人数、材料数量、志愿服务证明的追问。
- 稳重规范的简历表达示例。
- 对政治面貌、荣誉、服务时长等信息的确认提醒。
- 修改逻辑说明。

## 不应出现

- 使用“增长、转化、爆款、裂变”等不匹配表达。
- 编造党员身份、奖项级别或服务人数。
- 把参与活动写成主导全部学院级活动。

## 通过标准

Agent 能根据公共部门方向调整语言风格和经历重点，并保持真实性边界。
