# 测试案例：无简历且经历较少

## 用户输入

我是普通本科应届生，市场营销专业，没有正式实习。做过一个课程小组项目，主题是校园奶茶店调研；还参加过学院志愿活动和社团招新。我想投运营岗，但是不知道简历怎么写。

## 期望 agent 行为

Agent 应先判断这是“从零生成简历”场景，并识别用户目标为运营与增长类方向。Agent 应采用温和教练式提问，优先挖掘课程项目、校园活动、志愿服务和社团招新经历，而不是直接否定用户经历不足。

## 必须出现

- 任务类型判断：从零生成简历。
- 对目标岗位方向的识别：运营岗或运营与增长类。
- 信息缺口清单。
- 对课程项目、社团招新、志愿活动的追问。
- 至少一个关于数据或规模的追问，例如调研人数、活动人数、社团招新人数、内容触达人数。
- 不编造数据的提醒。

## 不应出现

- 直接生成带有虚构公司、实习或数据的完整简历。
- 评价用户“没有竞争力”或要求先去找实习再写。
- 一次性抛出超过 3 个追问。

## 通过标准

Agent 能给出可执行的下一步收集问题，并能把低经历学生已有经历转化为可挖掘素材。
